# FUKU-TABI 役割分担表

> このファイルはFUKU-TABIの開発・運用における役割と担当を定義します。

---

## 役割マップ

| 役割               | 担当              | 主な責務                                     |
| ------------------ | ----------------- | -------------------------------------------- |
| **ディレクター**   | 福田さん          | 要件定義・優先順位・最終承認                 |
| **UXデザイナー**   | Claude            | 画面動線・ワイヤーフレーム・ユーザー体験設計 |
| **UIデザイナー**   | Claude            | 配色・コンポーネント・Tailwind実装           |
| **コピーライター** | Claude            | サービス文言・LP・SEOテキスト                |
| **フロントエンド** | Claude            | React/Next.js・画面実装                      |
| **バックエンド**   | Claude            | API・DB設計・Supabase                        |
| **SEO担当**        | Claude            | メタ情報・キーワード・検索流入               |
| **セキュリティ**   | `/security`スキル | 脆弱性スキャン・認証設計                     |
| **QA**             | `/test`スキル     | テストコード・動作確認                       |
| **レビュアー**     | `/review`スキル   | コード品質・改善提案                         |
| **インフラ**       | Claude            | Vercel・ドメイン・パフォーマンス             |

---

## 福田さんの判断が必要な場面

以下は必ず福田さんに確認を取ってから進める：

- デザインの方向性（配色・雰囲気）の最終決定
- 料金・サービス内容の文言
- 外部サービスとの契約・連携
- 本番へのデプロイ

---

## 呼び出し方

```
/start [UXデザイナーとして] トップページの動線を設計する
/start [コピーライターとして] サービス紹介文を書く
/start [SEO担当として] メタ情報を整備する
/start [フロントエンドとして] 予約管理画面を実装する
```
